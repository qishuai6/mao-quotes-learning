# 毛主席语录学习工具

每日自动更新 10 条毛主席语录，结合 AI 精选职场智慧。

## 功能特点

- 📖 每日精选 10 条毛主席语录
- 🤖 集成 DeepSeek AI，智能筛选职场相关内容
- 🎨 扁平化中国红设计风格
- 📱 响应式设计，支持移动端
- ⬇️ 一键下载语录卡片
- ⌨️ 支持键盘左右键翻页

## 在线访问

点击下方按钮一键部署到 Netlify：

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/qishuai6/mao-quotes-learning)

## 本地运行

直接在浏览器中打开 `index.html` 文件即可。

## 技术栈

- 纯 HTML/CSS/JavaScript
- DeepSeek API
- html2canvas（截图功能）
